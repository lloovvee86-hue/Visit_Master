# Power Automate 설정 가이드 (쉐어포인트 엑셀 연동)

이 문서는 `index.html`에서 입력된 방문객 정보를 쉐어포인트 내의 엑셀 파일에 자동으로 기록하기 위한 **Power Automate 흐름(Flow)** 설정 방법을 안내합니다.

## 1. 사전 준비
- **쉐어포인트(SharePoint)** 내에 방문객 정보를 저장할 엑셀 파일(`방문객_기록.xlsx`)을 생성합니다.
- 엑셀 파일 내에 **표(Table)**를 삽입하고 다음 헤더를 작성합니다:
  - `성함`, `연락처`, `소속`, `방문목적`, `차량번호`, `식사여부`, `등록시간`

## 2. Power Automate 흐름 생성 단계

### 1단계: 트리거 설정 (HTTP 요청)
1. Power Automate에서 **내 흐름 > 새 흐름 > 즉시 클라우드 흐름**을 선택합니다.
2. **'HTTP 요청이 수신된 경우'** 트리거를 선택합니다.
3. **'샘플 페이로드를 사용하여 스키마 생성'**을 클릭하고 아래 JSON을 복사하여 붙여넣습니다:
```json
{
  "name": "홍길동",
  "contact": "010-0000-0000",
  "company": "풀무원",
  "purpose": "장비 점검",
  "carNumber": "12가3456",
  "meal": "이용함 (O)",
  "timestamp": "2026. 4. 16. 오전 10:00:00"
}
```

### 2단계: 엑셀 행 추가 액션 설정
1. **'새 단계'**를 클릭하고 **'Excel Online (Business) - 표에 행 추가'** 액션을 선택합니다.
2. **위치**, **문서 라이브러리**, **파일** 정보를 선택하여 미리 만든 `방문객_기록.xlsx`를 지정합니다.
3. **표**를 선택하면 자동으로 헤더 항목이 나타납니다.
4. 각 항목에 대응하는 **동적 콘텐츠**를 매핑합니다:
   - 성함: `name`
   - 연락처: `contact`
   - 소속: `company`
   - 방문목적: `purpose`
   - 차량번호: `carNumber`
   - 식사여부: `meal`
   - 등록시간: `timestamp`

### 3단계: URL 복사 및 코드 적용
1. 흐름을 **저장**하면 **'HTTP POST URL'**이 생성됩니다.
2. 생성된 URL을 복사합니다.
3. `index.html` 파일의 **61번 라인** 근처 `POWER_AUTOMATE_URL` 변수에 해당 URL을 붙여넣습니다.

## 3. 테스트 및 배포
- `index.html`을 브라우저에서 열고 실제 데이터를 입력하여 엑셀에 데이터가 들어오는지 확인합니다.
- 정상 작동 확인 후, `index.html` 주소를 QR 코드로 만들어 로비에 비치합니다.
